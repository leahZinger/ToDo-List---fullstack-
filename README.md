# ToDo List – Full Stack Project

## Description
A full-stack ToDo List application built as part of a Full Stack development project.  
The system includes a REST API backend and a React frontend.

## Technologies
### Backend
- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- JWT Authentication

### Frontend
- React
- JavaScript
- HTML & CSS

## Project Structure
- `TodoApi` – Backend (ASP.NET Core)
- `ToDoListReact` – Frontend (React)

## Running the Project
### Backend
1. Open the solution in Visual Studio / VS Code  
2. Update the database connection string if needed  
3. Run the API project  

### Frontend
1. Navigate to the `ToDoListReact` folder
